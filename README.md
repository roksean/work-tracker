# Work Tracker
Site to track what exactly I'm working on and for how much time

## API documentation

The backend exposes OpenAPI (Swagger) documentation. After starting the backend (e.g. `./gradlew bootRun` from `WorkTracker-Backend`), open:

- **Swagger UI:** [http://localhost:8080/swagger-ui.html](http://localhost:8080/swagger-ui.html) (or the port shown at startup)
- **OpenAPI JSON:** [http://localhost:8080/v3/api-docs](http://localhost:8080/v3/api-docs)

Swagger UI lists all endpoints, request/response schemas, and lets you try requests from the browser.
